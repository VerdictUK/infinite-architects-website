## üêõ PART 10: KNOWN ISSUES & FIXES

### Issue 1: Three.js CDN Slow

**Symptom:** Particles take long to appear
**Fix:** Self-host three.min.js or use faster CDN
```html
<!-- Alternative CDN -->
<script src="https://unpkg.com/three@0.128.0/build/three.min.js"></script>
```

### Issue 2: Fonts Flash

**Symptom:** Text flashes when fonts load
**Fix:** Add font-display: swap and preload
```html
<link rel="preload" href="font.woff2" as="font" crossorigin>
```

### Issue 3: Mobile Touch Delay

**Symptom:** Buttons feel slow on mobile
**Fix:** Add touch-action CSS
```css
button { touch-action: manipulation; }
```

### Issue 4: Loader Stuck

**Symptom:** Loader never completes
**Fix:** Check for JS errors in console; ensure fonts load

### Issue 5: Safari Animation Glitch

**Symptom:** Animations stutter in Safari
**Fix:** Use will-change and transform3d
```css
.animated { will-change: transform; transform: translateZ(0); }
```
